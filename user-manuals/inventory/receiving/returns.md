# Receiving Returns
The user can find the `Returns` option on the second number in the top left corner in the Receiving app below the facility name. On clicking it, the default page that will open will be the Returns View page. 

## Returns View page

The users can view and manage returns on this page. 


**Characteristics:**

* **Returns Information**: The Returns Page categorizes returns into three distinct categories:
  * Created
  * Received
  * Canceled

## Return Details page

The users can open the Return Details page for a specific return by clicking on the selected return. The Return Details page provides a detailed view of the selected return and the users can find information about the product that is being returned here.

### **Characteristics:** <a href="#key-characteristics" id="key-characteristics"></a>

* **Return Order Information:** The page shows the details of a specific return, including the return order ID, tracking number, status, date, and quantity.



<figure><img src="../.gitbook/assets/Return Details Page.png" alt=""><figcaption><p>Image: Return details page</p></figcaption></figure>



**Functions**

**a.  Search items:** Search returns by unique identifiers.


**b. Scan:** Scan to search the return item.


**c. Receive all:** Receive all the quantity of an item at once.


**d. Location:** Find the specific location where the return is being received in the facility.


**e. Input quantity:** Input the quantity of the item received in a return.


**f. Save function:** Once the return quantity has been received, complete the return-receiving process.
