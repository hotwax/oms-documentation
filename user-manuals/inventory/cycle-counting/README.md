# Cycle Count

## Introduction

HotWax Commerce's Cycle Count App enhances accuracy and efficiency in the product stock counting process. It helps store associates in reconciling systematic and physical inventory, and improve inventory accuracy. The users can scan products using their handheld devices ensuring swift and efficient counting process. 

{% hint style="info" %}
Users can log in to the Cycle Count app only if they are associated with a facility and only users with specific fulfillment permissions can conduct and review inventory counts. 
{% endhint %}

This section focuses on the various actions available in the Cycle Count app for the user to reconcile inventory. 


### List of pages in the Cycle Count app include:

1. Search Product page
2. Product Detail page
3. Upload Count page
4. Settings page
