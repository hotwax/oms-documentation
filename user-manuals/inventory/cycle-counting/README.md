# Cycle Count

## Introduction

HotWax Commerce's Cycle Count App enhances accuracy and efficiency in the product stock counting process. It helps store associates in reconciling systematic and physical inventory, and improve inventory accuracy. 

This section focuses on the various actions available in the Cycle Count app for the user to reconcile inventory. 


### List of pages in the Cycle Count app include:

1. Search Product page
2. Product Detail page
3. Upload Count page
4. Settings page
